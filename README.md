# 🎧 Spotify User Behaviour Dashboard

## 📊 Project Overview
This project presents an interactive Power BI dashboard that analyzes Spotify user behaviour. It provides insights into listening patterns, subscription types, user demographics, and device preferences.

## 🚀 Objectives
- Analyze user listening habits and engagement.
- Identify trends in music preferences and subscriptions.
- Visualize key metrics for data-driven decision-making.

## 📌 Key Insights
- Average Listening Hours per Week
- Average Skips per Day
- Subscription Type Distribution
- Primary Devices Used by Users
- Total Playlists Created
- Favorite Genres by Country
- Ad Conversion to Subscriptions
- Music Suggestion Ratings

## 🛠️ Tools & Technologies
- **Power BI** – Data Visualization
- **Microsoft Excel** – Data Processing
- **SQL (Optional)** – Data Analysis
